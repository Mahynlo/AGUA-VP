# Introducción al Módulo de Tarifas

En este módulo se administran las tarifas que usa el sistema para calcular cobros y se puede revisar cómo quedaría un consumo antes de usarlo en cobro o facturación.

La vista está pensada para trabajar de forma visual: primero se revisan las tarifas registradas, luego se agregan o corrigen si hace falta y, por último, se prueba un consumo en la calculadora.

## Qué incluye este módulo

- **Gestión de tarifas**: ver las tarifas registradas, buscar una tarifa y agregar una nueva.
- **Calculadora**: seleccionar una tarifa y probar un consumo para revisar el desglose.
- **Vigencia**: identificar cuáles tarifas siguen activas y cuáles están por vencer.

## Cómo usarlo

1. Abra la pestaña que necesita.
2. Revise primero la lista o la calculadora.
3. Si va a crear una tarifa, complete los datos y la vigencia.
4. Si solo quiere revisar un cobro, use la calculadora con un consumo de prueba.

## Recomendación

Use la calculadora antes de aplicar una tarifa nueva al trabajo diario. Así puede confirmar que el resultado visual coincide con lo que espera mostrar al usuario.
