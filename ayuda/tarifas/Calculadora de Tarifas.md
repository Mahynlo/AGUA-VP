# Calculadora de Tarifas

Esta sección sirve para probar un consumo y ver cómo quedaría el cobro antes de usarlo en una factura o revisión.

No está pensada para hacer cuentas manuales, sino para seleccionar una tarifa, ingresar un consumo y revisar el desglose que aparece en pantalla.

## Qué hace en pantalla

- Permite elegir una tarifa registrada.
- Acepta un consumo en metros cúbicos.
- Muestra el total calculado.
- Desglosa el cobro por tramos para que pueda revisarlo con facilidad.

## Cómo usarla

1. Abra la pestaña **Calculadora**.
2. Elija la tarifa que quiere probar.
3. Escriba el consumo en m³.
4. Presione **Calcular Desglose**.
5. Revise el total y la tabla con el detalle.

## Qué debe revisar en el resultado

- El consumo ingresado.
- El consumo facturable que muestra el sistema.
- El total calculado.
- Cada tramo que aparece en el desglose.

## Recomendación

Use esta vista cuando quiera confirmar rápidamente cuánto cobrará una tarifa antes de llevar el dato a otro proceso.
