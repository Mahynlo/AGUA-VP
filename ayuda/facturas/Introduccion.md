# Introducción al Módulo de Facturas

El módulo de **Facturas** le permite revisar, consultar y dar seguimiento a los cobros generados a partir de las lecturas.

Aquí puede ver qué facturas ya están emitidas, cuáles siguen pendientes y cuáles necesitan revisión.

---

## Qué verá en pantalla

- **Listado de facturas**: muestra cada cobro registrado.
- **Filtros de búsqueda**: ayudan a encontrar un cliente, un periodo o un estado específico.
- **Fechas de emisión y vencimiento**: facilitan el control de pagos.
- **Estados**: indican si la factura está pendiente, pagada o en revisión.

## Cómo trabaja el usuario con este módulo

1. Revise la lista para localizar la factura deseada.
2. Abra el detalle si necesita confirmar montos o fechas.
3. Verifique que el consumo y el importe coincidan con la lectura registrada.
4. Use el área de pagos cuando el cliente vaya a cubrir su cuenta.

## Relación con lecturas

Las facturas nacen de las lecturas capturadas.

- Si la lectura está bien, la factura se genera con normalidad.
- Si la lectura se corrige antes de facturar, el monto también se ajusta.
- Si hay un recalculo, el cobro reflejará el nuevo consumo.

> Esta pantalla es la referencia principal para saber cuánto debe cada cliente y en qué fecha se generó el cobro.
