# Asignar un Medidor

Asignar un medidor permite que el cliente quede listo para registrar lecturas y generar sus cobros.

## Pasos

1. Dirígete a la **tabla de clientes**.  
2. Selecciona el cliente al que deseas asignar un medidor.  
3. Haz clic en el botón con el **icono de lápiz ✏️**.  
4. Se abrirá un **modal de edición**, donde se mostrarán los datos actuales del cliente.  

Dentro de ese modal verá el apartado **Gestión de Medidor**.

- **Medidores asignados**: muestra los medidores que ya tiene ese cliente.
- **Buscar medidores**: permite encontrar uno por número de serie o referencia.
- **Disponibilidad**: el sistema le mostrará si el medidor se puede usar o no.

Si el medidor está libre, podrá seleccionarlo. Si ya está en uso, aparecerá marcado y no podrá elegirse para otro cliente.

---

### Confirmación de la asignación

Cuando elija el medidor correcto, presione **Actualizar Cliente** para guardar. Después de confirmar, el medidor quedará visible en la ficha del cliente.

> Si ve que el medidor ya está asignado a otra persona, debe liberarse antes de poder usarlo aquí.
