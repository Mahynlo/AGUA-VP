# Asignar un Medidor

Para asignar un **medidor** a un cliente existente, sigue estos pasos:

1. Dirígete a la **tabla de clientes**.  
2. Selecciona el cliente al que deseas asignar un medidor.  
3. Haz clic en el botón con el **icono de lápiz ✏️**.  
4. Se abrirá un **modal de edición**, donde se mostrarán los datos actuales del cliente.  

Dentro de este modal encontrarás el apartado **Gestión de Medidor**, que incluye:

- **Medidores asignados** → Muestra los medidores que ya están vinculados al cliente.  
- **Buscar medidores** → Permite localizar medidores disponibles o buscar uno en específico.  
- **Disponibilidad** → El sistema indicará si el medidor está disponible para asignación.  
   - Si el medidor está libre, podrás asignarlo inmediatamente.  
   - Si el medidor ya está en uso, aparecerá con la etiqueta **Asignado** y no podrá seleccionarse.  

---

### Confirmación de la asignación
Una vez seleccionado un medidor válido, confirma la operación haciendo clic en **Actualizar Cliente**.  
Si todo fue correcto, aparecerá una **notificación de éxito** y el medidor quedará registrado en la ficha del cliente.
