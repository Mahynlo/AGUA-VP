# Impresión de Recibos

Esta parte del módulo se usa cuando quiere preparar recibos para uno o varios clientes antes de mandarlos a imprimir.

La idea es sencilla: elija el periodo, seleccione los clientes que sí va a imprimir y revise primero la vista previa.

## Qué hace en pantalla

- Muestra la lista de clientes disponibles para ese periodo.
- Permite marcar uno o varios clientes.
- Deja cambiar el periodo antes de generar el documento.
- Abre una vista previa para revisar el recibo antes de enviarlo a impresión.

## Cómo usarlo

1. Abra la pestaña de impresión de recibos.
2. Verifique que el periodo sea el correcto.
3. Seleccione uno o varios clientes de la lista.
4. Presione **Vista previa** para revisar el resultado.
5. Si todo está correcto, use **Imprimir**.

## Qué conviene revisar

- Que el cliente tenga información completa.
- Que el periodo corresponda al cobro que quiere sacar.
- Que solo deje seleccionados los clientes que realmente necesita imprimir.

## Recomendación

Use la vista previa siempre que vaya a imprimir varios recibos. Eso le permite confirmar el formato antes de gastar papel.
