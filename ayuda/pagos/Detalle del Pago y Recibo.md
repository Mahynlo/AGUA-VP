# Detalle del Pago y Recibo

Esta vista muestra la información completa de un pago ya guardado.

## Qué puede revisar

- Folio del pago.
- Factura relacionada.
- Cliente que realizó el pago.
- Fecha del movimiento.
- Monto entregado y cambio.
- Usuario que registró el cobro.

## Para qué sirve

- Para confirmar que un pago quedó guardado correctamente.
- Para revisar el comprobante antes de entregarlo al cliente.
- Para imprimir o volver a abrir el recibo cuando haga falta.

## Cómo usarla

1. Abra el detalle del pago desde la lista.
2. Revise que el folio y la factura sean correctos.
3. Compruebe el monto y el método de pago.
4. Use la opción de **Ver / Imprimir Recibo** si necesita el comprobante.

## Recomendación

Después de un cobro, revise este detalle al menos una vez para asegurarse de que todo quedó asentado como esperaba.
