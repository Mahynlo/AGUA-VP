# Introducción al Módulo de Pagos

El módulo de **Pagos** le ayuda a registrar cuándo un cliente ya cubrió su deuda y a dejar constancia de la forma en que pagó.

También permite revisar pagos anteriores, comprobar saldos y mantener ordenado el historial de cobros.

---

## Qué verá en pantalla

- **Lista de pagos**: muestra los cobros ya registrados.
- **Búsqueda y filtros**: sirven para encontrar un pago por cliente, fecha o periodo.
- **Detalle del pago**: permite revisar cuánto se entregó, cuánto se cobró y si hubo cambio.
- **Cobranza por cliente**: muestra facturas, saldos, pagos y acceso al recibo desde una sola vista.
- **Pago rápido**: permite cobrar varios clientes o facturas de forma más ágil para un periodo específico.
- **Cartera vencida y cortes**: ayuda a ubicar clientes con deuda, crear convenios o aplicar cortes cuando corresponde.

## Cómo se usa

1. Busque la factura o cliente que desea cobrar.
2. Revise el importe pendiente.
3. Registre el pago con el método correcto.
4. Confirme que el movimiento haya quedado guardado.

Si el cliente no va a pagar todo en una sola vez, primero revise si conviene aplicar un pago parcial, un pago rápido del periodo o un convenio.

> Este módulo es la parte final del proceso: después de medir, facturar e imprimir, aquí se deja registrado el cobro.
