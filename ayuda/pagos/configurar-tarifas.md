# Configurar Tarifas

Esta sección sirve para entender cómo se calculan los cobros que luego aparecen en facturas y pagos.

## Para qué sirve

- Para saber cuánto se cobrará según el consumo.
- Para revisar los tramos de cobro antes de facturar.
- Para entender por qué una factura tiene cierto importe.

## Cómo leer el cálculo

- Primero existe un consumo base.
- Después, cada tramo de consumo puede tener un valor distinto.
- Mientras más agua use el cliente, mayor puede ser el total.

## Recomendación para el usuario

Antes de cobrar, revise que la lectura y el consumo coincidan con la realidad. Así la tarifa aplicada será la correcta.

> Si desea entender el detalle numérico, consulte también la guía del módulo de Tarifas.
