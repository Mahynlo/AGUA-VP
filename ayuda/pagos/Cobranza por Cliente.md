# Cobranza por Cliente

Esta vista sirve para revisar la deuda de cada cliente y cobrar desde su historial completo.

## Qué puede hacer aquí

- Ver todas las facturas del cliente.
- Revisar cuáles están pendientes y cuáles ya fueron pagadas.
- Registrar un pago parcial o completo.
- Abrir el detalle del pago para consultar el recibo.
- Imprimir el comprobante cuando sea necesario.

## Cómo usarla

1. Busque al cliente en la lista.
2. Abra su detalle de cobranza.
3. Revise el saldo, las facturas y el último pago registrado.
4. Elija si va a cobrar una factura, varias facturas o un monto parcial.
5. Confirme el pago y, si lo necesita, imprima el recibo.

## Cuándo conviene usarla

- Cuando el cliente quiere revisar toda su historia antes de pagar.
- Cuando necesita aplicar un pago distribuido entre varias facturas.
- Cuando desea comprobar cuánto queda pendiente después de un cobro parcial.

> Esta vista es útil cuando no solo le interesa registrar un pago, sino también entender cómo quedó la deuda del cliente.
