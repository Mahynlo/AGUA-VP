# Pago Rápido

El pago rápido sirve para registrar cobros de un periodo completo de forma más ágil, sin entrar factura por factura.

## Cuándo conviene usarlo

- Cuando quiere cobrar varias facturas de un mismo periodo.
- Cuando el cliente ya está listo para liquidar de forma masiva.
- Cuando desea acelerar el trabajo en una jornada con muchos pagos.

## Cómo funciona

1. Abra la opción de **Modo Rápido**.
2. Seleccione el periodo que va a cobrar.
3. Revise la lista de facturas que aparecerán.
4. Marque las que no se van a pagar ahora, si aplica.
5. El sistema procesará el resto de los cobros.

## Qué debe revisar antes de confirmar

- Que el periodo sea el correcto.
- Que no esté dejando fuera una factura que sí debía incluirse.
- Que el método de pago corresponda al cobro realizado.

> Úselo cuando quiera ahorrar tiempo, pero sin dejar de revisar los datos antes de confirmar.
