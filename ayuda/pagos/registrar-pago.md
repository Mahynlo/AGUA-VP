# Registrar Pago

Registrar un pago significa dejar anotado que el cliente ya entregó el dinero correspondiente a su factura.

## Pasos

1. Abra el módulo **Pagos**.
2. Busque la factura o el cliente que va a pagar.
3. Presione la opción para registrar un nuevo pago.
4. Capture el monto recibido.
5. Seleccione la forma de pago.
6. Guarde la operación.

## Qué debe revisar antes de confirmar

- Que la factura sea la correcta.
- Que el monto recibido coincida con lo que el cliente entregó.
- Que el método de pago sea el adecuado.
- Que la fecha del pago esté correcta.

## Cómo se entiende el cobro

- Si el cliente paga el total, la factura quedará cubierta.
- Si entrega más dinero del necesario, el sistema mostrará el cambio.
- Si entrega menos, quedará saldo pendiente para después.

## Después de guardar

- Revise el detalle del pago para ver el folio y el recibo.
- Si el cliente necesita comprobante, use la opción de ver o imprimir.
- Si el cobro fue parcial, el saldo restante seguirá visible en el historial.

> Registre el pago exactamente como se recibió para que el historial del cliente quede claro.
