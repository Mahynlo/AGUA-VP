# Cartera Vencida y Cortes

Esta sección se usa para revisar clientes con deuda y decidir si conviene crear un convenio, hacer un corte o reconectar el servicio.

## Qué verá

- Clientes con saldo pendiente.
- Nivel de atraso o gravedad de la deuda.
- Estado de corte o convenio.
- Acciones para crear convenio, registrar corte o reconexión.

## Cómo trabajar con ella

1. Abra la lista de deudores.
2. Revise el nivel de atraso y el saldo pendiente.
3. Si el cliente puede ponerse al corriente, cree un convenio.
4. Si ya no hay forma de regularizar al momento, revise la opción de corte.
5. Cuando el cliente se ponga al día, use la reconexión.

## Recomendación

- Antes de cortar, confirme que realmente no hay un convenio activo.
- Si el cliente ya está en convenio, revise el saldo del acuerdo antes de tomar otra acción.
- Use la impresión o el detalle para dejar constancia del movimiento.

> Esta parte es delicada: sirve para tomar decisiones de cobro, no solo para ver números.
