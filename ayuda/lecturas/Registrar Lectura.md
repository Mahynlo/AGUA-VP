# Registrar una Lectura

Registrar una lectura significa anotar cuánto marcó el medidor en esa visita.

## Pasos

1. Abra el módulo **Lecturas**.
2. Entre al apartado de **Registro de Lecturas**.
3. Busque el medidor que le toca revisar.
4. Escriba la lectura observada.
5. Confirme y guarde.

## Qué debe revisar antes de guardar

- Que el número del medidor corresponda al cliente correcto.
- Que la lectura sea mayor o igual a la anterior, salvo que el sistema indique otra situación.
- Que la fecha y la ruta sean las correctas.

## Cómo entender el consumo

El sistema usa la lectura actual y la lectura anterior para mostrar el consumo estimado en metros cúbicos.

- Si la diferencia es pequeña, el consumo será bajo.
- Si la diferencia es alta, el consumo también subirá.
- Si el valor no parece correcto, conviene revisarlo antes de guardar.

> Capture primero, revise después y solo confirme cuando esté seguro.
