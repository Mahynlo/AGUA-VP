# Introducción al Módulo de Lecturas

El módulo de **Lecturas** es donde se registra el consumo real de cada medidor y se da seguimiento a la ruta de trabajo.

Aquí puede ver qué lecturas faltan, cuáles ya fueron capturadas y cuáles necesitan corregirse antes de facturarse.

---

## Qué muestra la pantalla

- **Registro de lecturas**: lista de rutas y medidores pendientes.
- **Métricas y reportes**: resumen del avance y resultados del trabajo.
- **Botones de acción**: permiten registrar, revisar, corregir e imprimir.
- **Facturación de ruta**: genera las facturas del periodo cuando la ruta ya quedó completa.
- **Recalcular facturación**: permite ajustar el cálculo de una ruta que ya fue facturada, cuando todavía corresponde corregirla.

## Cómo funciona para el usuario

1. Seleccione la ruta o el grupo de medidores que le corresponde.
2. Capture la lectura que observa en el equipo.
3. Revise que el consumo calculado tenga sentido.
4. Guarde la lectura para que pueda usarse después en la factura.

## Cuándo conviene revisar una lectura

- Si el consumo se ve demasiado alto o demasiado bajo.
- Si el medidor fue cambiado recientemente.
- Si el dato original tuvo un error al capturarse.
- Si la ruta ya fue facturada y necesita una corrección antes de cerrar el periodo.

> Este módulo es importante porque de aquí sale el consumo que después se cobra al cliente.
